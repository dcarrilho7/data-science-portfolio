## Projects

### `rl_gridworld`
Q-learning agent trained on a custom 2D GridWorld with traps and a goal state.

**How to run**
- From the repo root:
  - `python -m venv .venv`
  - `source .venv/bin/activate`
  - `pip install -r requirements.txt`
  - `python rl_gridworld/train.py`

**Optional flags**
- `--episodes 5000`
- `--epsilon 0.1`
- `--seed 42`
- `--no-plot`
